# Simple-App

Simple-App - приложение для анализа содержимого веб-страниц, подсчета слов и 
количества их появления на странице

## Запуск

Запуск приложения осуществляется командой
```sh
java -jar simple-app-1.0-jar-with-dependencies.jar
```

Для автоматического анализа веб-страниц их адреса можно указать в командной строке 
при запуске программы, например:

```sh
java -jar simple-app-1.0-jar-with-dependencies.jar https://google.ru https://google.com
```

## Руководство пользователя

При запуске прораммы укажите необходимые адреса веб-страниц как аргументы командной строки,
разделяя их пробелами. Программа завершит работу после обработки всех адресов.

Если при запуске не будут указаны адреса, программа предложит ввести их вручную:

```sh
Please enter URL (e.g. https://google.com/) or enter "q" to exit
```

Пример вводимого адреса:

```sh
https://google.com/
```

Для выхода из программы ввести

```sh
q
```
